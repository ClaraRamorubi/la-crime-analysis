# Analyzing Crime Patterns in Los Angeles

👩🏽‍💻 Author: Clara Ramorubi  
Aspiring Data Analyst | Python & Excel

## 📌 Project Overview
This project analyzes crime data from Los Angeles to uncover patterns in crime occurrence based on time, location, and victim demographics. The goal is to derive actionable insights using data analysis and visualization techniques in Python.

## 🛠️ Tools Used
- Python
- pandas
- matplotlib
- seaborn

## 📊 Key Analysis

### 1. Crime Frequency by Hour
- Converted time data into hourly format
- Visualized crime frequency using a countplot
- Identified the hour with the highest number of crimes

### 2. Night Crime Analysis
- Filtered crimes committed between 10 PM and 3:59 AM
- Determined which area had the highest number of night crimes

### 3. Victim Age Group Analysis
- Grouped victims into age categories
- Counted the number of crimes per age group

## 🔍 Key Insights
- Crime frequency varies significantly by hour
- Certain areas experience higher crime rates at night
- Some age groups are more frequently affected by crime
